Estudando Tabela Hash

O que são tabelas hash?

Uma tabela hash, de dispersão ou espalhamento é uma estrutura
de dados especial, que associa chaves de pesquisa a valores.

Uma tabela hash é um generalização da idéia de array, porém
utiliza uma função denominada HASHING para espalhar os elementos
fazendo com que os mesmos fiquem de forma não ordenada dentro
do "array" que define a tabela.
