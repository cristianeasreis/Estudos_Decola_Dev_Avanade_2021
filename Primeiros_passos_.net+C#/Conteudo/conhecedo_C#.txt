O que é e como funciona o C#

O que é o C# ?

É uma linguagem que começou na decada de 90.
C# é uma linguagem elegante, orientada a objeto e fortemente
tipada.
A sintaxe do C# é similar a do C, C++ Ou Java.

Suporta os conceitos de encapsulamento, herança e
polimorfismo(OO).

Os programs C# são executados no .Net, que incluio.

.CLR (Common Language Runtime)
.Conjunto unificado de bibliotecas de classes.

Atualmente o compilador do C# é o Roslyn.

Como Funciona ?

O Código-fonte escrito em C# é compilado em uma linguagem
intermediária(IL).

O código e os recursos de IL são armazenados no disco
em um arquivo executável chamado assembly geralmente
com uma extensão .exe o .dll

Quando o programa C# é executado, o assembly é carregado
no CLR.
Em seguida o CLR executará a compilação Just in Time(JIT)
para converter o codigo IL em intruçoes de maquin nativas.

O CLR também fornece outros serviços:

.GARBAGE COLLECTOR
.EXCEPTION HANDLER
.RESSOURCES MANAGER

Além dos serviços de tempo de execução , o .Net tambem
inclui uma extensa biblioteca com milhares de classes que 
fornecem uma ampla variedade de funcionalidades uteis, desde
entrada e saida de arquivos, manipulação de cadeias de carcteres
analise XML , etc...
